# Kaoooai product website

Static multi-page website for the Japan local B2B market.

## Pages

- `index.html` - product homepage
- `about.html` - product concept and target organizations
- `features.html` - platform overview, AI agent functions, product characteristics, review AI, customization, and planned agents
- `effect.html` - quantitative effect, ROI calculator, and PoC evaluation points
- `security.html` - security and compliance
- `flow.html` - PoC, customization, onboarding, and rollout flow
- `cases.html` - anonymous case studies and planned agent areas
- `articles.html` - article hub for releases, blogs, anonymous cases, and guides
- `article/release-kaoooai.html` - product announcement press release
- `article/case-sier.html` - anonymous case study for SIer / contract development
- `article/case-insourcing.html` - anonymous case study for in-house product development
- `article/case-qa.html` - anonymous case study for QA / testing teams
- `article/ai-development-standardization.html` - blog article
- `article/poc-interview-template.html` - PoC / interview article template
- `docs.html` - technical documentation and configuration overview
- `resources.html` - downloads and FAQ
- `contact.html` - demo / PoC consultation form
- `sitemap.html` - review sitemap
- `sitemap.txt` - plain text version of the sitemap

Open `index.html` directly in a browser, or run a local static server from this folder.
